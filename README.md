# About Comics
An app to get information about movies.



# Application Features
- Search for movies;
- Get information about movies;
- Bookmark favorite books;
- Dark/Light mode theme.




# Used Libraries
- Dagger Hilt
- Coroutines
- Navigation Component
- Room
- Paging 3
- Retrofit
- Picasso




# Architecture
- MVVM with View Binding
